HTML

HyperText Markup Language - язык разметки веб-страниц и приложений.

Типы тегов (парные / одиночные | строчные / блочные)

одиночные - img, input, hr, br
парные - всё, что осталось
блочные - занимают всю ширину, которая есть (например: div, p, h1-h6 + html5 tags)
строчные - по ширине контента (например: a, strong, em, u, span)

  • Строчные (inline) — span, em, strong
  • Блочные (block) — div, p, article
  • Строчно-блочные (inline-block) — input, textarea

Основные теги

<a>
<a href="https://example.com" target="_blank">Пример ссылки</a> - это ссылка с атрибутом <href>, значением атрибута https://example.com и содержанием Пример ссылки. Ссылкой можно сделать изображение <a href="#"><img src="#" alt="#"></a>
<img>
Тег картинки <img src="https://example.com/logo.png" alt="Logo"> - это пример одиночного тега, в котором не используется закрывающий тег.
<ol> <ul> <li>
Списки - пример тегов, которые состят из нескольких вложенных элементов.
<ol> (ordered list) пронумерованный список.
<ul> (unordered list) непронумерованный список.
<li> (list item) элемент списка.

<ol>
<li>Первый элемент</li>
<li>Второй элемент</li>
<li>Третий элемент</li>
<ol>
<table> <tr> <td>
Границы и внешний вид таблиц настраивать через CSS. Использовать моноширинные шрифты.

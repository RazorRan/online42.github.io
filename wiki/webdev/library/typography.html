Нестандартные шрифты

TTF и OTF — форматы с минимальным сжатием или без него.

WOFF и WOFF2 — форматы c высокой степенью сжатия.

WOFF2 — лучший выбор, для подстраховки используется совместно с WOFF, для поддержки устаревших браузеров требуются TTF и OTF.

Подключение через @font-face

@font-face {
font-family: 'Ubuntu';
font-style: normal;
font-weight: 400;
font-display: swap;
src:
url('../fonts/Ubuntu.woff2') format('woff2'),
url('../fonts/Ubuntu.woff') format('woff');
}
  • первым в src:url должен быть самый подходящий формат, а далее — запасные варианты
  • для каждого начертания нужно отдельное правило @font-face
  • для улучшения производительности правило @font-face лучше всего прописывать в самом начале CSS-файла
  • с помощью функции local можно загружать шрифт с компьютера пользователя, если он установлен, но отображаение может быть некорректным

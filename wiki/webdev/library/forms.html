Кнопки в формах

<input type="image" src="img/otpravit.png" alt="Отправить"> - кнопка отправки формы в виде картинки
<input type="submit" value="Отправить"> - кнопка отправки формы через input
<button type="submit">Отправить</button> - кнопка отправки формы

type="submit" - отправляет форму на сервер
type="reset" - сбрасывает введенные в форму данные
type="button" - убирает функциональность с тега button для работы с кнопкой через JavaScript
